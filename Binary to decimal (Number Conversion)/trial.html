Binary to Decimal (Number Conversion)

Problem Statement:

Given a Binary number in interger format. Output the equivalent Decimal number.

Illustration:

Example 1:

Binary Number: 1010

Explanation: The binary number 1010 has 4 digits. Each digit is multiplied by the corresponding power of 2 and summed:
1x(2^3) + 0x(2^2) + 1x(2^1) + 0x(2^0) = 8 + 0 + 2 + 0 = 10. Therefore, the decimal equivalent of 1010 is 10.

Example 2:

Binary Number: 11011

Explanation: The binary number 11011 has 5 digits. Each digit is multiplied by the corresponding power of 2 and summed:
1x(2^4) + 1x(2^3) + 0x(2^2) + 1x(2^1) + 1x(2^0) = 16 + 8 + 0 + 2 + 1 = 27. Therefore, the decimal equivalent of 11011 is 27.

Prerequisites:

To understand and implement the solution, you should be familiar with the following concepts:

  • Arithmetic Operations (%, *, +, /)
  • Looping Constructs (while loop)
  • Conditional Statements (if-else)

Solution:

Classic C Program:


#include <stdio.h>
#include <stdlib.h>

int convert(long long int bin)
{
    int sum = 0, p = 1;
    int n;

    while (bin)
    {
        n = bin%10;

        if (n<=1)
        {
            sum = sum + (n*p);
            p = p*2;
            bin = bin/10;
        }
        else
        {
            printf("The number is not in binary Format!!!");
            return -1;
        }

    }
    return sum;
}

int main()
{
    long long int bin;

    printf("Enter Number in Binary : ");
    scanf("%lld", &bin);

    if (convert(bin)!= -1)
    {
        printf("\nThe decimal equivalent of %lld is : %d\n", bin, convert(bin));
    }
    else
    {
        printf("\nNumbers other than 0 and 1 are present\n");
    }
    return 0;
}

Python Solution :


def convert(bin):
    sum = 0
    p = 1

    while bin:
        n = bin % 10

        if n <= 1:
            sum += n * p
            p *= 2
            bin //= 10
        else:
            print("The number is not in binary Format!!!")
            return -1

    return sum

if __name__ == "__main__":
    bin = int(input("Enter Number in Binary : "))

    result = convert(bin)

    if result != -1:
        print("\nThe decimal equivalent of", bin, "is :", result)
    else:
        print("\nNumbers other than 0 and 1 are present")

Explanation:

1. The program prompts the user to input a binary number.
2. Inside the `convert` function, it initializes variables `sum` and `p` to store the decimal equivalent and the power of 2 respectively.
3. Using a while loop, the program iterates through each digit of the binary number.
4. Within the loop, it extracts the last digit of the binary number using modulo operation.
5. If the extracted digit is less than or equal to 1, it multiplies the digit by the appropriate power of 2 and adds the result to the sum.
6. The power of 2 (`p`) is updated for the next iteration.
7. After processing all digits, it returns the decimal equivalent if the binary number contains only 0s and 1s.
8. If any digit other than 0 or 1 is encountered, the program prints a message indicating that the number is not in binary format.
