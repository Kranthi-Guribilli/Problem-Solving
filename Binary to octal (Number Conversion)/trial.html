Binary to Octal (Number Conversion)

Problem Statement:

Given a Binary number in interger format. Output the equivalent Octal number.

Illustration:

Example:

Binary Number: 101110101

Explanation: Starting from the rightmost side, we group the binary digits into sets of three: (101) (110) (101). Converting each group of three binary digits to its octal equivalent:
- (101) in binary is equivalent to (5) in octal.
- (110) in binary is equivalent to (6) in octal.
- (101) in binary is equivalent to (5) in octal.
Concatenating the octal representations: 565.
Therefore, the octal equivalent of the binary number 101110101 is 565.

Solution:

Classic C Program:


#include <stdio.h>
#include <stdlib.h>

int octal(int bin)
{
    int n, sum=0, p=1;
    int octarr[] = {0, 1, 10, 11, 100, 101, 110, 111};

    while (bin)
    {
        n = bin%1000;

        for (int i = 0; i<8; ++i)
        {
            if(octarr[i]==n)
            {
                sum = sum + i*p;
                break;
            }
        }
        bin = bin/1000;
        p = p*10;
    }
    return sum;
}

Python Solution :


def octal(bin):
    octarr = {0: 0, 1: 1, 10: 2, 11: 3, 100: 4, 101: 5, 110: 6, 111: 7}
    sum = 0
    p = 1

    while bin:
        n = bin % 1000
        for key, value in octarr.items():
            if key == n:
                sum += value * p
                break
        bin //= 1000
        p *= 10

    return sum

if __name__ == "__main__":
    bin = int(input("Enter your Binary Number : "))
    print("The Octal Equivalent of", bin, "is :", octal(bin))

Explanation:

1. The program prompts the user to input a binary number.
2. Inside the `octal` function (or equivalent), it initializes a dictionary (C: array) containing the octal equivalents of all possible 3-bit binary patterns.
3. Using a while loop, the program iterates through the binary number, grouping the binary digits into sets of three from right to left.
4. Within the loop, it looks up each group of three binary digits in the dictionary (C: array) to find its corresponding octal value.
5. The octal digits are then multiplied by the appropriate power of 10 and summed to form the final octal equivalent.
6. After processing all groups of three binary digits, the resulting octal number is returned.
7. Finally, the program prints the octal equivalent of the input binary number.
