Upper Triangular Check

Problem Statement:

Write a program to determine whether a given matrix is an upper triangular matrix.

An upper triangular matrix is a square matrix in which all elements below the main diagonal are zero.

Illustration:

1 2 3
0 4 5
0 0 6

The matrix is an upper triangular matrix.

1 2 3
4 5 5
0 6 7

The matrix is not an upper triangular matrix.

Prerequisites:

To understand and implement the solution, you should be familiar with the following concepts:

  • Basic matrix manipulation
  • Nested loops
  • Conditional statements

Solution:

Classic C Program:


#include <stdio.h>
#include <stdlib.h>
#define MAXROW 10
#define MAXCOL 10

void input_arr(int arr[][MAXCOL], int m, int n);

int main()
{
    int arr[MAXROW][MAXCOL];
    int nrow, ncol;
    int i, j;
    int upper = 1;

    printf("Enter number of Rows : ");
    scanf("%d", &nrow);
    printf("Enter number of Colums : ");
    scanf("%d", &ncol);

    printf("Enter array elements :\n");
    input_arr(arr, nrow, ncol);

    for (i=0; i<nrow; ++i)
    {
        for (j=0; j<ncol; ++j)
        {
            if (i>j)
            {
                if (arr[i][j] != 0)
                {
                    upper = 0;
                    break;
                }
            }
        }
        if (upper == 0)
            break;
    }

    if (upper == 1)
        printf("\nThis is a Upper Triangular Matrix\n");
    else
        printf("\nThis is not an Upper Triangular Matrix\n");

    return 0;
}

void input_arr(int arr[][MAXCOL], int m, int n)
{
    int i, j;

    for (i=0; i<m; ++i)
    {
        for (j=0; j<n; ++j)
        {
            scanf("%d", &arr[i][j]);
        }
    }
    return;
}

Python Solution :


def input_arr(arr, m, n):
    for i in range(m):
        for j in range(n):
            arr[i][j] = int(input())

def is_upper_triangular(arr, m, n):
    for i in range(m):
        for j in range(n):
            if i > j and arr[i][j] != 0:
                return False
    return True

def main():
    MAXROW = 10
    MAXCOL = 10

    arr = [[0] * MAXCOL for _ in range(MAXROW)]

    nrow = int(input("Enter number of Rows: "))
    ncol = int(input("Enter number of Columns: "))

    print("Enter array elements:")
    input_arr(arr, nrow, ncol)

    if is_upper_triangular(arr, nrow, ncol):
        print("\nThis is an Upper Triangular Matrix")
    else:
        print("\nThis is not an Upper Triangular Matrix")

if __name__ == "__main__":
    main()

Explanation:

1. The program starts by prompting the user to input the number of rows and columns for the matrix.
2. It then prompts the user to input the elements of the matrix.
3. The input_arr function is responsible for accepting the matrix elements and storing them in a 2D array.
4. The main function iterates over each element of the matrix and checks if it lies below the main diagonal (i > j) and is not equal to zero.
5. If any such element is found, the matrix is not an upper triangular matrix, and the program breaks out of the loop.
6. If no such element is found, the matrix is considered an upper triangular matrix.
7. Finally, the program prints the appropriate message based on whether the matrix is upper triangular or not.

Time and Space Complexity:

The time complexity of the input_arr function is O(m * n), where m is the number of rows and n is the number of columns.
This is because the function iterates over each element of the matrix once.
The time complexity of the is_upper_triangular function is also O(m * n), as it iterates over each element of the matrix once to check if it lies below the main diagonal.
Overall, the time complexity of the program is O(m * n), where m is the number of rows and n is the number of columns.

The space complexity of the program is O(m * n) as well. This is because the program uses a 2D array of size m * n to store the matrix elements.
Additionally, the program uses a few constant extra variables and functions, so they do not significantly contribute to the overall space complexity.
Therefore, the dominant factor in the space complexity is the size of the matrix itself.
