left-aligned inverted right triangle pattern

Problem Statement:

Write a C program that generates a left-aligned inverted right triangle pattern '#' symbols.

Illustration:

Enter no of rows : 5

Output Pattern:

# # # # #
# # # #
# # #
# #
#

Prerequisites:

To understand and implement the solution, you should be familiar with the following concepts:

  • Basics of loops, Nested loops

Solution:

Classic C Program:


#include <stdio.h>
#include <stdlib.h>

/*

# # # # #
# # # #
# # #
# #
#

*/

int main()
{
    int row, i, col, j;

    printf("Enter number of rows : ");
    scanf("%d", &row);

    for (i = 1; i <= row; i++){
        for (j = 1; j <= (row - i + 1); j++){
            printf("$ ");

        }
        printf("\n");
    }

    return 0;
}

Python Solution :


def main():
    row = int(input("Enter number of rows: "))
    for i in range(1, row + 1):
        for j in range(1, row - i + 2):
            print("#", end=" ")
        print()

if __name__ == "__main__":
    main()

Explanation:

1. User Input: Accept the number of rows for the pattern from the user.

2. Outer Loop (Rows): Iterate from 1 to the given number of rows.

3. Inner Loop (Columns): Iterate from 1 to (row - i + 1) for each row.

4. The number of symbols in each row decreases by 1 as you move down the rows.

5. Print Symbol: Print the "#" symbol for each column.

6. Move to Next Line: After printing all symbols in a row, move to the next line.

j = 0 j = 1 j = 2 j = 3
i = 0 # # # #
i = 1 # # #
i = 2 # #
i = 3 #

Time and Space Complexity:

Time Complexity: O(n^2) - where "n" is the number of rows. This is because there are nested loops iterating over each element of the pattern.

Space Complexity: O(1) - There is no extra space used that depends on the input size.
