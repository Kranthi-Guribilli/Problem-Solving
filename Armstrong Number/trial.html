Armstong Number

Problem Statement:

Write a program to determine whether a given integer is an Armstrong number. An Armstrong number (also known as a narcissistic number or plenary number) is a number that is equal to the sum of its own digits raised to the power of the number of digits.

Illustration:

Consider the following examples:

Example 1:

Number: 153

Explanation: The number 153 has 3 digits. Each digit raised to the power of 3 and then summed gives: 1^3 + 5^3 + 3^3 = 1 + 125 + 27 = 153. Since the sum equals the original number, 153 is an Armstrong number.

Example 2:

Number: 123

Explanation: The number 123 has 3 digits. Each digit raised to the power of 3 and then summed gives: 1^3 + 2^3 + 3^3 = 1 + 8 + 27 = 36. Since the sum does not equal the original number (123), 123 is not an Armstrong number.

Prerequisites:

To understand and implement the solution, you should be familiar with the following concepts:

  • Simple Arithmeteic Operations(%, /, +)
  • Mathematical funtions (pow(), log10())
  • loops
  • Conditional statements

Solution:

Classic C Program:


#include <stdio.h>
#include <stdlib.h>
#include <math.h>

int main()
{
    int num, dnum, digits, rem, sum;
    sum = 0;

    printf("Enter a number to check : ");
    scanf("%d", &num);
    dnum = num;

    digits = (int)(log10(num)+1);

    while (num)
    {
        rem = num%10;
        sum = sum + pow(rem, digits);
        num = num/10;
    }

    if (dnum == sum)
    {
        printf("Armstrong number\n");
    }
    else
    {
        printf("Not Armstrong number\n");
    }

    return 0;
}

Python Solution :


def is_armstrong(num):
    dnum = num
    digits = len(str(num))
    sum = 0

    while num > 0:
        rem = num % 10
        sum += rem ** digits
        num //= 10

    if dnum == sum:
        print("Armstrong number")
    else:
        print("Not Armstrong number")

if __name__ == "__main__":
    num = int(input("Enter a number to check: "))
    is_armstrong(num)

Explanation:

1. The program prompts the user to input a number to check.
2. It calculates the number of digits in the input number using the logarithm function and adds 1 to account for the first digit.
3. It initializes variables to store the sum of powered digits and a copy of the original number.
4. Using a while loop, the program iterates through each digit of the number.
5. Inside the loop, it calculates the remainder when dividing the number by 10 to extract the last digit.
6. It raises this digit to the power of the total number of digits and adds the result to the sum.
7. After processing all digits, it compares the sum with the original number.
8. If they are equal, it concludes that the number is an Armstrong number and prints a corresponding message. Otherwise, it prints that the number is not an Armstrong number.

Time and Space Complexity:

The time complexity of the given logic is O(log10(n)), where n is the input number. This is because the number of iterations in the while loop is determined by the number of digits in the input number, which is logarithmic with respect to the value of the number.

The space complexity of the logic is O(1), as the space required remains constant regardless of the input size. The space complexity is constant because the program uses a fixed number of variables and does not allocate any additional memory based on the input size.
