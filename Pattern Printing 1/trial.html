Square Grid

Problem Statement:

Write a C program that generates a square grid pattern with '#' symbols.

Illustration:

Enter no of rows : 4

Output Pattern:

# # # #
# # # #
# # # #
# # # #

Prerequisites:

To understand and implement the solution, you should be familiar with the following concepts:

  • Basics of loops, Nested loops

Solution:

Classic C Program:


#include <stdio.h>
#include <stdlib.h>

/*
# # # #
# # # #
# # # #
# # # #
*/

int main()
{
    int row, i, col, j;
    printf("Enter number of rows : ");
    scanf("%d", &row);

    col = row;

    for (i = 1; i <= row; i++)
    {
        for (j = 1; j <= col; j++)
        {
            printf("# ");
        }
        printf("\n");
    }

    return 0;
}

Python Solution :


def square_grid(n):
    for i in range(n):
        for j in range(n):
            print("# ", end="")
        print()

row = int(input("Enter number of rows: "))
square_grid(row)

Explanation:

1. It utilizes a nested loop structure to iterate over each row and column of the grid.

2. The outer loop, controlled by the variable `i`, iterates over the rows of the grid.

3. Within each row, the inner loop, controlled by the variable `j`, iterates over the columns of the grid.

4. Inside the inner loop, the function prints the character `"# "` to represent each cell of the grid.

5. After printing all the characters in a row, the function moves to the next line, effectively shifting to the next row.

6. This process repeats for each row until the entire grid is printed.

j = 0 j = 1 j = 2 j = 3
i = 0 # # # #
i = 1 # # # #
i = 2 # # # #
i = 3 # # # #

Time and Space Complexity:

Time Complexity: O(n^2), where n is the number of rows (or columns) in the grid. This is because the program iterates over each cell in the grid using nested loops.

Space Complexity: O(1), as the program only uses a constant amount of memory for variables regardless of the grid size.

Challenge:

An additional challenge could be to modify the program to allow the user to specify the symbol used for the grid (e.g., '*', '+', '-', etc.) and customize the grid's size and appearance.
