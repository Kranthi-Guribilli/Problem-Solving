left-aligned right triangle pattern

Problem Statement:

Write a C program that generates a left-aligned right triangle pattern '#' symbols.

Illustration:

Enter no of rows : 5

Output Pattern:

#
# #
# # #
# # # #
# # # # #

Prerequisites:

To understand and implement the solution, you should be familiar with the following concepts:

  • Basics of loops, Nested loops

Solution:

Classic C Program:


#include <stdio.h>
#include <stdlib.h>

/*

#
# #
# # #
# # # #
# # # # #

*/

int main(){
    int row, i, col, j;

    printf("Enter number of rows : ");
    scanf("%d", &row);

    for (i = 1; i <= row; i++){
        for (j=1; j <= i; j++){
            printf("# ");
        }
        printf("\n");
    }
    return 0;
}

Python Solution :


row = int(input("Enter number of rows : "))

for i in range(1, row + 1):
    for j in range(1, i + 1):
        print("# ", end="")
    print()

Explanation:

1. Prompt the user to enter the number of rows for the pattern.

2. Use nested loops to iterate over each row and column.

3. In the outer loop, iterate over the rows from 1 to the user-specified number.

4. In the inner loop, iterate over the columns from 1 to the current row number.

5. Print '#' symbol for each column.

6. After printing '#' symbols for each row, move to the next line.

j = 0 j = 1 j = 2 j = 3
i = 0 #
i = 1 # #
i = 2 # # #
i = 3 # # # #

Time and Space Complexity:

Time Complexity: O(n^2), where n is the number of rows. This is because there are nested loops to print each '#' symbol.

Space Complexity: O(1), as the space required is constant, independent of the input size.

Challenge:

An additional challenge could be to modify the program to allow the user to specify the symbol used for the grid (e.g., '*', '+', '-', etc.) and customize the grid's size and appearance.
