nCr Using Pascal's triangle

Problem Statement:

Implement a Recursive program to calculate the combination nCr using Pascal's triangle.

Illustration:

Input: User enters a binary number, for example, "1010".

Explanation: To convert binary to decimal:

  • Start from the rightmost digit (LSB).
  • Each digit is multiplied by 2 raised to the power of its position (0-based index).
  • Add all these results to get the decimal equivalent.
For "1010":
  • 0 * 2^0 = 0
  • 1 * 2^1 = 2
  • 0 * 2^2 = 0
  • 1 * 2^3 = 8
Sum = 0 + 2 + 0 + 8 = 10

Output: The decimal equivalent is 10.

Prerequisites:

To understand and implement the solution, you should be familiar with the following concepts:

  • Basic arithmetic operations.
  • Concept of binary numbers.
  • loops and conditional statements.

Solution:

Classic C Program:


#include <stdio.h>
#include <stdlib.h>
#include <math.h>

int main()
{
    long long bin;
    int isbin, n, sum, rem;
    
    printf("Enter a number in binary format : ");
    scanf("%lld", &bin);
    sum = 0;
    n = 0;
    isbin = 1;
    while (bin)
    {
        rem = bin % 10;
    
        if ((rem == 0) || (rem == 1))
        {
            sum = sum + (pow(2, n)*rem);
            bin = bin/10;
            n++;
        }
        else
        {
            printf("You have entered invalid number\n");
            isbin = 0;
            break;
        }
    }
    
    if (isbin == 1)
    {
        printf("Decimal number is : %d\n", sum);
    }
    
    return 0;
}

Python Solution :


def main():
    bin_number = int(input("Enter a number in binary format: "))
    decimal_number = 0
    n = 0
    is_binary = True
    
    while bin_number:
        rem = bin_number % 10

        if rem == 0 or rem == 1:
            decimal_number += (2 ** n) * rem
            bin_number //= 10
            n += 1
        else:
            print("You have entered an invalid number")
            is_binary = False
            break

    if is_binary:
        print("Decimal number is:", decimal_number)

if __name__ == "__main__":
    main()

Explanation:

1. The program prompts the user to input a binary number.

2. It initializes variables to store the decimal equivalent (`decimal_number`), the current power of 2 (`n`), and a flag to indicate if the input is binary (`is_binary`).

3. It iterates over each digit of the binary number using a while loop.

4. For each digit, it checks if it's either 0 or 1. If it's not, it sets the `is_binary` flag to False and breaks out of the loop.

5. If the digit is 0 or 1, it calculates the decimal equivalent by multiplying the digit with 2 raised to the power of its position (`2 ** n`) and adds it to the `decimal_number`.

6. After processing each digit, it updates the binary number by integer dividing it by 10 to remove the processed digit and increments the power `n`.

7. If the input is a valid binary number (`is_binary == True`), it prints the decimal equivalent. Otherwise, it displays an error message.

Time and Space Complexity:

Time Complexity: The time complexity of the given algorithm is O(log N), where N is the input binary number. This is because the number of iterations required to convert a binary number to decimal is proportional to the number of digits in the binary number, which is logarithmic.

Space Complexity: The space complexity is O(1), as the algorithm only requires a constant amount of additional space for variables regardless of the input size.
