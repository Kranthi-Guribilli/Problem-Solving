Right Triangle Pattern with Odd Numbers

Problem Statement:

Write a C program that generates a Right Triangle Pattern with Odd Numbers. Each row contains consecutive odd numbers starting from 1.

Illustration:

Enter no of rows : 5

Output Pattern:

1
1 3
1 3 5
1 3 5 7
1 3 5 7 9

Prerequisites:

To understand and implement the solution, you should be familiar with the following concepts:

  • Basics of loops, Nested loops

Solution:

Classic C Program:


#include <stdio.h>
#include <stdlib.h>

/*

1
1 3
1 3 5
1 3 5 7
1 3 5 7 9

*/

int main()
{
    int row, i, col, j;

    printf("Enter number of rows : ");
    scanf("%d", &row);

    for (i = 1; i <= row; i++){
        for (j = 1; j <= i; j++){
            printf("%d ", ((2*j)-1));
        }
        printf("\n");
    }

    return 0;
}

Python Solution :


def generate_pattern(rows):
    for i in range(1, rows + 1):
        for j in range(1, i + 1):
            print(2 * j - 1, end=" ")
        print()

rows = int(input("Enter number of rows: "))
generate_pattern(rows)

Explanation:

1. Initialize a variable row to store the number of rows.

2. Prompt the user to enter the number of rows.

3. Use nested loops to iterate through each row and column:
Outer loop (i) iterates through each row from 1 to row.
Inner loop (j) iterates through each column in the current row (i) from 1 to i.

4. In the inner loop, calculate the value to print using the formula 2 * j - 1, which generates consecutive odd numbers starting from 1.

5. Print the calculated value followed by a space.

6. After printing all values for the current row, move to the next line.

7. Repeat steps 4-6 for each row until the pattern is complete.

j = 0 j = 1 j = 2 j = 3 j = 4
i = 0 1
i = 1 1 3
i = 2 1 3 5
i = 3 1 3 5 7
i = 4 1 3 5 7 9

Time and Space Complexity:

Time Complexity: O(n^2) - where "n" is the number of rows. This is because there are nested loops iterating over each element of the pattern.

Space Complexity: O(1) - There is no extra space used that depends on the input size.
