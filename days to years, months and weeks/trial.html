Days to Years, Months, Weeks and days

Problem Statement:

Given a number of days, convert it into years, months, weeks, and days.

What to do??

Input:
Enter no of days : 1000

Output:
YEARS : 2
MONTHS : 8
WEEKS : 4
DAYS : 2

1000 days can be divided into 2 years, 8 months, 4 weeks, and 2 days.

Prerequisites:

To understand and implement the solution, you should be familiar with the following concepts:

  • Basic arithmetic operations.

Solution:

Classic C Program:


#include <stdio.h>
#include <stdlib.h>

int main()
{
    int days, years, months, weeks;

    printf("Enter no of days : ");
    scanf("%d", &days);

    years = (days/365);
    months = (days - (years * 365))/30;
    weeks = (days - ((years * 365)+(months * 30)))/7;
    days = (days - ((years*365)+(months * 30)+(weeks*7)));

    printf("YEARS : %d\n", years);
    printf("MONTHS : %d\n", months);
    printf("WEEKS : %d\n", weeks);
    printf("DAYS : %d\n", days);

    return 0;
}

Python Solution :


def main():
    days = int(input("Enter no of days : "))

    years = days // 365
    months = (days - (years * 365)) // 30
    weeks = (days - ((years * 365) + (months * 30))) // 7
    days = days - ((years * 365) + (months * 30) + (weeks * 7))

    print("YEARS :", years)
    print("MONTHS :", months)
    print("WEEKS :", weeks)
    print("DAYS :", days)

if __name__ == "__main__":
    main()

Explanation:

1 Prompt the user to input the number of days.

2 Calculate the number of years by dividing the total number of days by 365.

3 Calculate the number of months by subtracting the days equivalent to the number of years and then dividing the remaining days by 30.

4 Calculate the number of weeks by subtracting the days equivalent to the number of years and months, and then dividing the remaining days by 7.

5 Calculate the remaining days by subtracting the days equivalent to the number of years, months, and weeks.

6 Output the calculated years, months, weeks, and days.

for no of days = 1000, they days can be divided into 2 years, 8 months, 4 weeks, and 2 days.
- Each year contains 365 days, so 1000 days correspond to 2 years.
- After subtracting 2 years (730 days), the remaining days are 270.
- Each month contains approximately 30 days, so 270 days correspond to 8 months.
- After subtracting 8 months (240 days), the remaining days are 30.
- Each week contains 7 days, so 30 days correspond to 4 weeks.
- After subtracting 4 weeks (28 days), the remaining days are 2, which represents the final days.

Time Complexity:O(1) because the number of calculations remains constant regardless of the input size.
