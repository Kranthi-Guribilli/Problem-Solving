Palindrome Check

Problem Statement:

Write a program to determine whether a given string is a palindrome.

A palindrome is a word, phrase, number, or other sequence of characters that reads the same forwards and backwards, ignoring spaces, punctuation, and capitalization.

Illustration:

Input: "radar"

Explanation: The given string "radar" is a palindrome because it reads the same forwards and backwards.

Output:- The string is a palindrome.

Input: "hello"

Explanation: The given string "hello" is not a palindrome because it does not read the same forwards and backwards.
- Characters at start and end indices are not equal ('h' != 'o')

Output:- The string is not a palindrome.

Prerequisites:

To understand and implement the solution, you should be familiar with the following concepts:

  • Basic string manipulation
  • Looping constructs
  • Conditional statements

Solution:

Classic C Program:


#include <stdio.h>
#include <stdlib.h>

int main()
{
    char str[100];
    int len, startindex, endindex;

    printf("Enter your string : ");
    gets(str);

    len = 0;
    while(str[len] != '\0') len++;

    startindex = 0;
    endindex = (len-1);

    while (startindex <= endindex)
    {
        if (str[startindex] != str[endindex])
            break;

        startindex++;
        endindex--;
    }

    if (startindex >= endindex)
        printf("The string is Palindrome!\n");
    else
        printf("The string is not Palindrome!\n");


    return 0;
}

Python Solution :


def check_palindrome(s):
    start_index = 0
    end_index = len(s) - 1

    while start_index <= end_index:
        if s[start_index] != s[end_index]:
            return False
        start_index += 1
        end_index -= 1

    return True

def main():
    s = input("Enter your string: ")

    if check_palindrome(s):
        print("The string is Palindrome!")
    else:
        print("The string is not Palindrome!")

if __name__ == "__main__":
    main()

Explanation:

1. The program starts by prompting the user to input a string.
2. It calculates the length of the input string by iterating through it until it encounters the null terminator ('\0').
3. Two index variables, `startindex` and `endindex`, are initialized to the beginning and end of the string, respectively.
4. Using a while loop, the program checks if the characters at `startindex` and `endindex` are equal.
5. If the characters are not equal, the loop breaks, indicating that the string is not a palindrome.
6. If the loop completes without breaking, it means that all characters were compared successfully, confirming that the string is a palindrome.
7. The program then prints "The string is Palindrome!" if the condition `startindex >= endindex` is true, indicating that the string is indeed a palindrome.
8. Otherwise, it prints "The string is not Palindrome!".
9. Finally, the program terminates.

Time and Space Complexity:

The time complexity of this program is O(n), where n is the length of the input string. This is because the program iterates through half of the string in the worst case, comparing characters from both ends.

The space complexity is O(1) or constant. The program uses a fixed amount of memory for variables
