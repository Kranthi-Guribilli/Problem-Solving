Key in every segment

Problem Statement:

You are given an array of integers, a key value, and a segment size. Your task is to determine if the key value exists in every non-overlapping segment of the specified size in the array.
Write a program that takes the following inputs:
1. The size of the array.
2. The elements of the array.
3. The key value to search for.
4. The segment size.
The program should output "Yes" if the key value exists in every non-overlapping segment of size k in the array, otherwise "NO".

Illustration:

Example:

Array: [4, 2, 1, 5, 3, 6, 8, 7]

Key: 3

Segment Size: 3

Explanation: We divide the array into non-overlapping segments of size 3: [4, 2, 1], [5, 3, 6], [8, 7].
- In the first segment [4, 2, 1], the key 3 is not present.
- In the second segment [5, 3, 6], the key 3 is present.
- In the third segment [8, 7], the key 3 is not present.
Since the key is not present in every non-overlapping segment, the output is "NO".

Solution:

Classic C Program:


#include <stdio.h>
#include <stdlib.h>

int main() {
    int i, j, n, k, x, count;
    int *arr;

    printf("Enter the size of array : ");
    scanf("%d", &n);

    arr = (int *)malloc(sizeof(int)*n);

    printf("Enter array elements : ");

    for(i=0; i<n; ++i)
        scanf("%d", &arr[i]);

    printf("Enter key : ");
    scanf("%d", &x);

    printf("Enter segment size : ");
    scanf("%d", &k);

    for(i=0; i<n; i+=k)
    {
        count = 0;
        for(j=i; j<min(i+k, n); ++j)
        {
            if(arr[j]==x)
            {
                count = 1;
                break;
            }
        }

        if(count == 0)
        {
            printf("NO");
            break;
        }
    }

    if(count == 1)
        printf("Yes");

    return 0;
}

Python Solution :


def main():
    n = int(input("Enter the size of array : "))
    arr = list(map(int, input("Enter array elements : ").split()))
    x = int(input("Enter key : "))
    k = int(input("Enter segment size : "))

    for i in range(0, n, k):
        count = 0
        for j in range(i, min(i + k, n)):
            if arr[j] == x:
                count = 1
                break

        if count == 0:
            print("NO")
            break

    if count == 1:
        print("Yes")

if __name__ == "__main__":
    main()

Explanation:

1. The program starts by prompting the user to input the size of the array and the array elements.
2. It then dynamically allocates memory to store the array elements based on the user-provided size.
3. Next, the program asks the user to input the key value to search for and the segment size.
4. Using nested loops, the program iterates through the array, dividing it into non-overlapping segments of the specified size.
5. Within each segment, another loop iterates through the elements to search for the key value.
6. If the key value is found in any segment, a flag is set to indicate its presence.
7. After searching through all segments, the program checks the flag to determine if the key value was found in every segment.
8. Depending on the flag status, the program prints "Yes" if the key value is found in every segment, otherwise it prints "NO".
9. Finally, the program deallocates the memory allocated for the array and terminates.

Time and Space Complexity:

The program iterates through each element of the array once to search for the key value within each segment. Within each segment, it iterates through a subset of elements (of size k) to perform the search. Since the nested loops are not nested to the full size of the array but instead to the size of the segments, the overall time complexity is still O(n), where n is the size of the array.

The program dynamically allocates memory for the array to store the elements entered by the user. Therefore, the space complexity of the program is O(n), where n is the size of the array.
